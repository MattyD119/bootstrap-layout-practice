# TinDog Bootstrap Project

This project was completed as part of a Bootstrap learning section.  
The goal was to **recreate a reference website layout** using Bootstrap’s grid system, components, and utility classes.

## Project Overview

TinDog is a playful concept site showcasing a dating app for dogs.  
The design includes:

- A hero section with a call-to-action
- Feature sections with icons and descriptions
- Pricing plans for different “dog profiles”
- Responsive layout for mobile and desktop

## What was provided
- Partial HTML structure
- Gradient background CSS
- Reference images and SVG icons for design guidance

## My focus
- Implementing **responsive layouts** using Bootstrap
- Using **Bootstrap components** like navbars, cards, buttons, and utility classes
- Replicating the reference design while understanding how classes affect layout and spacing
- Ensuring **visual consistency** across devices

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5

## Notes
This project is for **learning purposes** and demonstrates Bootstrap fundamentals, including grid usage, responsive design, and component implementation.
